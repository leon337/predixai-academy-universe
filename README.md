# PredixAI Academy Universe

Universo didático oficial da PredixAI Academy.

Este repositório centraliza empresas, personagens, cenários, sistemas, documentos e histórias fictícias utilizados de forma contínua nos cursos, manuais e projetos educacionais.

## Objetivo

Manter um contexto compartilhado, coerente e evolutivo para exemplos, analogias, exercícios, estudos de caso, simulações e projetos integradores.

## Tipos de informação

### Canônica

Informação oficialmente aprovada e obrigatória para todos os cursos e agentes de IA que utilizem este universo. Alterações exigem decisão registrada.

### Não canônica

Rascunhos, hipóteses, ideias e experimentos ainda não aprovados. Não devem ser tratados como fatos oficiais do universo.
